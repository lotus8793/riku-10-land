# りくの10ランド

4歳向けの「10になるなかよしペア」練習アプリです。

## Macで試す

このフォルダの `index.html` をブラウザで開くと遊べます。

PWAやホーム画面追加に近い形で試す場合は、ObsidianSyncのルートで次を実行します。

```sh
python3 -m http.server 8097
```

その後、ブラウザで次を開きます。

```text
http://localhost:8097/Projects/riku-10-land/index.html
```

## iPhoneで試す

MacとiPhoneが同じWi-Fiにいる状態で、MacのローカルIPアドレスを使って開きます。

```text
http://<MacのIPアドレス>:8097/Projects/riku-10-land/index.html
```

Safariで開けたら、共有ボタンから「ホーム画面に追加」を選びます。
