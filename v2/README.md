# りくのぼうけん（v2）

v1（親フォルダ）とは独立した魔改造版。localStorageキーは `riku10v2-*` でv1と分離。

## モード（6タブ）

| タブ | 内容 |
|------|------|
| しゅぎょう | 答えが9以下の足し算（36問） |
| あわせて10 | 10の補数ペア |
| ぼうけん | さくらんぼ計算。正解で赤四角が10マスへ飛んで10が完成 |
| ひきざん | 10以内の引き算（45問）。✕で消えるアニメーション |
| ずかん | ポケモン図鑑 |
| きろく | 60秒チャレンジのモード別ランキング |

## ポケモンシステム

- 「じかんあり」の正解10問ごとにゲットチャンス（全モード累計）
- ボール演出: 揺れる → 金色=ゲット（80%）/ はじけて逃げられる（20%）
- 色違い10%。演出中はタイマー停止、「つぎへ」で再開
- きょうのミッション: 1日20問正解でボーナス1匹（確定）。タブ下にプログレスバー
- 図鑑: 60種。重複は×Nバッジ、色違い持ちは✨。所持数順ソート、未ゲットは？
- 「ぜんぶにがす」で図鑑と累計をリセット
- じかんなしモードは練習用（ゲット対象外）

## 演出

- 正解で紙吹雪＋ほめことば6種ランダム＋コンボ表示（🔥Nれんぞく）
- 正解音はアルペジオ。コンボ5ごとに1音ずつ豪華に（最大7音）

## 注意

- ポケモン画像はPokéAPI（公式アートワーク）から取得。初回はオンライン必須、以降はキャッシュ
- ポケモンは任天堂の著作物のため**ネット公開・配布はしないこと**（家庭内利用のみ）

## 試し方

`index.html` をブラウザで開く。PWAはObsidianSyncルートで:

```sh
python3 -m http.server 8097
```

→ `http://localhost:8097/Projects/riku-10-land/v2/index.html`

iPhoneは同じWi-FiでMacのIPを使い、Safari共有→「ホーム画面に追加」（モンスターボールのアイコンになる）。
